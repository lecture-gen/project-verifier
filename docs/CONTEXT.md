## 용어 사전 (Glossary)

| 용어 | 정의 |
|------|------|
| **Artifact** | 학생이 제출한 프로젝트 산출물. ZIP 파일 또는 GitHub 리포지토리 |
| **Evaluation** | 하나의 평가 세션. Artifact 제출부터 최종 리포트 생성까지의 전체 단위 |
| **Ingestion** | Artifact를 파싱하고 벡터 인덱싱하는 전처리 단계 |
| **Interview** | AI가 학생에게 Artifact 근거 기반 질문을 하고 답변을 받는 대화 세션 |
| **Question** | Bloom's Taxonomy 레벨이 지정된 개별 면접 질문 |
| **Turn** | Interview 내 하나의 질문-답변 쌍 |
| **Rubric** | 답변 채점 기준. 각 Question에 대해 AI가 생성 |
| **Score** | 개별 Turn에 대한 채점 결과 |
| **Report** | Evaluation 종료 후 생성되는 영역별 진위 검증 리포트 |
| **Area** | 프로젝트를 구성하는 기술/역할 영역 (예: 백엔드, 프론트엔드, 인프라) |
| **Bloom's Level** | 질문 난이도 분류: Remember → Understand → Apply → Analyze → Evaluate → Create |

## 알려진 과제

- **LLM 처리 시간**: 각 단계(분석→질문생성→인터뷰→리포트)에서 AI 호출 시간이 길어 라이브 시연 가능 여부 불확실
- **LLM 비용**: OpenAI API 호출 비용 누적
- **채점 정확도 검증**: LLM이 생성한 채점 결과를 정량적으로 검증할 기준/로직 부재 — 가장 취약한 부분
